# Experiment Results

This directory is reserved for verified experiment results generated by the DS-3DCNN training and evaluation pipelines.

The original clinical MRI dataset and patient-level predictions are not included in this repository.

## Expected Training Outputs

Running:

```bash
python -m src.train --config configs/default.yaml
```

generates the following files inside the configured output directory:

```text
outputs/
├── best_fold_1.pth
├── best_fold_2.pth
├── best_fold_3.pth
├── best_fold_4.pth
├── best_fold_5.pth
│
├── fold_1_results.json
├── fold_2_results.json
├── fold_3_results.json
├── fold_4_results.json
├── fold_5_results.json
│
├── fold_1_loss.png
├── fold_1_accuracy.png
├── fold_1_f1.png
├── fold_1_confusion_matrix.png
│
├── cross_validation_results.csv
├── cross_validation_summary.json
└── summed_confusion_matrix.png
```

Similar plots are generated for every cross-validation fold.

## Cross-Validation Results

The file:

```text
cross_validation_results.csv
```

contains one row for each fold with:

- fold number;
- best training epoch;
- best validation macro F1-score;
- best validation loss;
- test loss;
- accuracy;
- macro precision;
- macro recall;
- macro F1-score;
- multiclass ROC-AUC.

## Cross-Validation Summary

The file:

```text
cross_validation_summary.json
```

contains the mean and standard deviation across all folds for:

- accuracy;
- macro precision;
- macro recall;
- macro F1-score;
- multiclass ROC-AUC.

It also contains the summed confusion matrix across all five test folds.

## Evaluation Outputs

Running:

```bash
python -m src.evaluate \
  --config configs/default.yaml \
  --checkpoint outputs/best_fold_1.pth
```

creates:

```text
outputs/evaluation/
├── best_fold_1_evaluation.json
└── best_fold_1_confusion_matrix.png
```

## Grad-CAM Outputs

Running:

```bash
python -m src.gradcam \
  --config configs/default.yaml \
  --checkpoint outputs/best_fold_1.pth \
  --image /path/to/anonymized_scan.nii.gz \
  --target-class PD \
  --output-prefix outputs/gradcam_pd
```

creates:

```text
outputs/gradcam_pd.nii.gz
outputs/gradcam_pd.png
```

The NIfTI file contains the complete three-dimensional Grad-CAM activation map.

The PNG file contains a center-slice visualization of the MRI volume and Grad-CAM overlay.

## Result Verification

Only results produced by the documented code and verified experiment configuration should be added to this repository.

Do not upload:

- fabricated plots;
- manually edited metrics;
- patient identifiers;
- restricted clinical metadata;
- MRI volumes;
- patient-level predictions containing identifiable information;
- unverified performance claims.

## Reported Research Performance

The PhD experiment reported approximately:

| Metric | Reported value |
|---|---:|
| Accuracy | 0.91 ± 0.02 |
| Macro precision | 0.92 |
| Macro recall | 0.91 |
| Macro F1-score | 0.91 |
| Macro ROC-AUC | 0.93 |

These values should be replaced or supplemented with the original verified output files when available.

Exact results depend on:

- the MRI cohort;
- preprocessing pipeline;
- data splitting;
- software environment;
- hardware;
- random seed;
- training configuration.
